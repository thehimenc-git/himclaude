💡 힘클로드란?
🤖 힘클로드가 하는 일

힘클로드는 더힘이엔씨(주)의 AI 업무보고 자동화 시스템입니다. 직원이 오늘 한 일을 입력하면 AI가 자동으로 내용을 정리하고, 매일 아침 전직원 이메일로 레벨별 요약 보고서가 자동으로 전송됩니다.

✅ 직원이 할 일
  • 힘클로드 HTML 파일 열기
  • 이름 선택
  • 업무 내용 텍스트로 입력
  • 제출 버튼 클릭

→ 이게 전부입니다!

⚙️ 자동으로 되는 것
  • AI가 내용 자동 정리·분류
  • 보고서 자동 생성
  • 전직원 이메일 자동 발송
  • 28일치 이력 누적 관리
ℹ️
현재 지원 입력 형식: 텍스트만 가능
이미지, 오디오 파일, PDF 등의 파일 첨부는 현재 지원되지 않으며, 차후 코드가 수정된 후 입력이 가능할 예정입니다.
📝 힘클로드 사용 방법
🖥️ STEP 1 — 힘클로드 파일 열기

담당자에게 전달받은 힘클로드 HTML 파일을 더블클릭해서 웹브라우저로 열어주세요.

※ 별도 설치 필요 없음. 인터넷 브라우저(크롬, 엣지 등)에서 바로 열립니다.

✏️ STEP 3 — 업무 내용 입력하기

텍스트 입력창에 오늘 진행한 업무 내용을 자유롭게 입력하세요. 형식 없이 자연스럽게 쓰시면 됩니다.

입력 예시

오늘 양동숲길 보행로 조성사업 관련 담당부서 연락처 확인 완료했고, 제기동 타당성 조사 보고서 수정 중입니다. 내일까지 완성 예정이고 CJ가양 설계도서 취합도 진행했습니다. 내일 10시에 용산전자상가 회의 있습니다.

🚀 STEP 4 — 제출하기

내용 입력 후 제출 버튼을 클릭하면 자동으로 전송됩니다.

✅ 제출 완료 메시지가 뜨면 정상 전송된 것입니다. 이후 모든 처리는 자동으로 이루어집니다.

  • → 다음날 보고서 참고 이력으로 포함됨
⚠️ 보고서가 자동 생성되는 오전 8:50~9:00 사이에는 제출을 잠시 피해주세요.
💬 아직 완벽하지 않습니다. 오류나 문의사항이 있으면 담당자에게 문의해 주세요.
⚠️ 주의사항 및 자주 묻는 질문
⚠️ 꼭 지켜주세요
  • 반드시 본인 이름을 선택하고 제출하세요.
  • 현재는 텍스트 입력만 가능합니다. 이미지·오디오·PDF 파일 첨부는 차후 지원 예정입니다.
  • 보고 마감은 매일 오전 08:50입니다. 그 이후 제출은 다음날 보고서에 반영됩니다.
  • 제출 완료 메시지가 뜨지 않으면 전송이 안 된 것입니다. 다시 시도해주세요.
  • 내용은 간단해도 됩니다. 핵심 업무 내용만 자연스럽게 입력하면 AI가 알아서 정리합니다.
❓ 자주 묻는 질문
제출했는데 확인이 안 돼요. 잘 전송된 건가요? ▼
제출 후 화면에 완료 메시지가 뜨면 정상 전송된 것입니다. 완료 메시지가 뜨지 않으면 인터넷 연결을 확인하고 다시 시도해주세요. 같은 내용을 두 번 제출해도 시스템이 자동으로 중복을 제거하므로 걱정하지 않아도 됩니다.
보고서를 잘못 제출하면 어떻게 되나요? ▼
잘못 제출하더라도 수정본을 다시 제출하시면 됩니다. 다만 다시 제출하실 때 아래와 같이 한 줄 추가해주세요.

예시: "예전 보고서가 잘못돼 다시 제출합니다. (이하 정정 내용 입력)"
이렇게 입력하면 AI가 이전 보고와 구분하여 수정 내용을 보고서에 반영합니다.
퇴근 후 집에서 제출해도 되나요? ▼
됩니다. 전날 오후 4시 이후부터 당일 오전 8:50 이전이면 언제 어디서든 제출 가능합니다. 퇴근 후 귀가해서 제출하거나, 다음날 출근 전 아침에 제출해도 당일 보고서에 포함됩니다.
내 이름이 목록에 없어요. ▼
담당자에게 연락해주세요. 직원 목록에 이름을 추가해드립니다. 이름이 없는 상태로 제출하면 보고서에 반영되지 않습니다.
어떤 내용을 써야 하나요? 형식이 있나요? ▼
정해진 형식 없습니다. 오늘 진행한 업무, 미팅 내용, 이슈, 내일 일정 등을 자유롭게 적으시면 됩니다. AI가 자동으로 정리해줍니다. 짧게 써도 됩니다.
이미지나 파일을 첨부하고 싶어요. ▼
현재는 텍스트 입력만 지원합니다. 이미지, 오디오 파일, PDF 등의 파일 첨부 기능은 차후 코드가 수정된 후 지원될 예정입니다. 현재는 파일 내용을 텍스트로 요약해서 입력해주세요.
보고서는 언제, 어떻게 받나요? ▼
매일 오전 8:50 이후 자동으로 생성되어 전직원 등록 이메일로 발송됩니다. 본인 등급(L1~L4)에 해당하는 보고서가 HTML 파일 첨부 형태로 이메일로 옵니다.
