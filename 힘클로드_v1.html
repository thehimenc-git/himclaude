✦Claude 힘보고 셋업

1회 셋업 후로는 모든 도구가 자동으로 본인 시트에 매핑됩니다.

1. MCP 커넥터 URL

claude.ai 프로젝트 → 커넥터 → 사용자 정의 MCP 서버 → 위 URL 붙여넣기

2. 프로젝트 지침 생성

[결과 복사]로 프로젝트 지침란에 붙여넣거나, [txt 저장]으로 받은 파일을 프로젝트에 자료로 첨부하세요.

🎙음성 후처리 셋업

메일로 받은 음성 텍스트를 정제 transcript로 다듬는 전용 프로젝트 지침을 만듭니다.

1. MCP 커넥터 URL (일일보고와 동일 커넥터)

후처리 전용 프로젝트 → 커넥터 → 사용자 정의 MCP 서버 → 위 URL 붙여넣기 (get_staff·get_context·get_personal·remember 사용)

2. 후처리 지침 생성

전용 프로젝트를 따로 만들 필요 없이, [txt 저장]으로 받은 파일을 일일보고와 같은 프로젝트에 자료로 첨부해도 됩니다. 후처리할 땐 "첨부된 음성후처리지침 따라줘"로 시작하세요. (지침란에 직접 붙여넣으려면 [결과 복사])

📦통합 셋업 — 박스 + txt 4종

이름 한 번 입력 → 프로젝트 지침란에 넣을 박스와, 자료로 첨부할 txt 4개를 만듭니다.

1. MCP 커넥터 URL

claude.ai 프로젝트 → 커넥터 → 사용자 정의 MCP 서버 → 위 URL 붙여넣기

2. 지침 생성

① 박스 지침 — 복사해서 프로젝트 지침란에 붙여넣기

② 지침 txt 4종 — 받아서 프로젝트 자료(지식)에 첨부

✦셋업 선택

어떤 셋업을 진행할지 선택하세요.
